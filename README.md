# Terraform AWS Starter

A reusable Terraform starter project for provisioning AWS infrastructure across multiple environments.

## Features
- Modular Terraform structure
- Environment separation (dev, qa, prod)
- VPC-ready layout
- S3 backend ready
- Scalable for ECS, Lambda, and RDS expansion

## Structure
- modules/
- env/dev
- env/qa
- env/prod

## Future Enhancements
- ECS service deployment
- RDS PostgreSQL
- Application Load Balancer
- CloudWatch monitoring
